# 🌸 Iris Flower Classification using Machine Learning

A responsive web application built with **Flask** and **Machine Learning** that predicts the species of an Iris flower based on its sepal and petal measurements.

The application uses a trained **Random Forest Classifier** to classify flowers into one of the following species:

- Iris-setosa
- Iris-versicolor
- Iris-virginica

---

## 📌 Features

- 🌸 Predict Iris flower species instantly
- 🤖 Machine Learning powered prediction
- 📊 Displays prediction confidence
- 💻 Responsive and modern user interface
- ⚡ Fast predictions using Flask
- 🎨 Beautiful glassmorphism-inspired design
- 📱 Mobile-friendly layout

## 🚀 Installation

### Clone the repository

```bash
git clone https://github.com/yourusername/Iris_Flower_Classification.git
```

### Navigate to the project folder

```bash
cd Iris_Flower_Classification
```

### Install dependencies

```bash
pip install -r requirements.txt
```

### Run the application

```bash
python app.py
```

Open your browser and visit:

```
http://127.0.0.1:5000/
```
